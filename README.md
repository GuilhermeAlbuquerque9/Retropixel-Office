# 💻 Retropixel Office™

Um pacote de aplicativos de escritório leve, rápido e com personalidade — inspirado na experiência dos anos 1990/2000.

---

## ✨ Sobre

O **Retropixel Office™** é uma suíte de produtividade que prioriza:

* Simplicidade
* Desempenho
* Experiência de uso agradável

Sem dependências desnecessárias, sem serviços externos e sem complexidade excessiva.

Tudo roda diretamente no navegador — e os arquivos são seus.

---

## 📦 Aplicativos incluídos

* 📝 **Write** — editor de texto
* 📊 **Sheets** — planilhas com suporte a fórmulas
* 🎞 **Slides** — apresentações com modo tela cheia
* ✅ **Tasks** — gerenciador de tarefas

---

## 💾 Arquivos

Cada aplicativo utiliza seu próprio formato:

| Aplicativo | Extensão |
| ---------- | -------- |
| Write      | `.rdoc`  |
| Sheets     | `.rshe`  |
| Slides     | `.rshow` |
| Tasks      | `.rtask` |

Os arquivos são salvos diretamente no seu computador.
